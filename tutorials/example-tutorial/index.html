Introduction

This is an example tutorial page. You can customize this content with your own tutorial information, code examples, images, and more.

To create a new tutorial, simply:

  1. Create a new folder in the tutorials/ directory
  2. Copy this index.html file and customize it
  3. Create an assets/ folder inside your tutorial folder
  4. Update the tutorial-script.js file with your assets
  5. Add your asset files to the assets/ folder
